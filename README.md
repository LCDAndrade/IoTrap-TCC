# IoTrap - Resultados Preliminares (Fase 1)

Repositório com os materiais complementares, diagramas e códigos-fonte do Trabalho de Conclusão de Curso.

# Códigos e Automação
Neste repositório encontram-se os scripts responsáveis por:
- Extração de IoCs (Indicadores de Comprometimento) dos logs do honeypot Cowrie.
- Integração com as APIs do VirusTotal e GeoIP.
- Criação de regras de bloqueio dinâmico no firewall (iptables).

# Arquitetura e Resultados Analíticos
Os diagramas de fluxo de dados e as visualizações geradas na Stack ELK (Kibana), que evidenciam a volumetria de ataques capturados durante a homologação, encontram-se disponíveis nos ficheiros de imagem anexados a este repositório.
